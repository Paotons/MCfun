📦 项目概览

MCfun 是一个运行在移动端的完整 Minecraft 指令开发环境。从项目创建到智能编辑、语法解析、行为包导出，全流程闭环。

📁 脚本文件: 131 个
📝 代码总量: ~11000 行
🧩 场景文件: 14 个
🔧 全局单例: 3 个

⚙️ 指令类型体系

NORMAL
普通 Minecraft 指令
NATIVE (&)
本地宏指令，编译时展开
&tp · &fill · &scoreboard
COMMENT (@)
注解指令，元编程支持
@list
HELP (?)
帮助指令

🧩 Element 继承树（25+ 类）

Element
├─ BoolElement · IntElement · FloatElement · StringElement · WordElement
├─ CoordElement · SelectorElement · SpaceItemElement
├─ BacketElement → ParamBacket · EqualParamBacket · ArrayBacket
└─ MultiParamElement → CoordsElement · PointPathElement

🎯 核心亮点

🔥 语法热加载
JSON 定义指令规则，编译后缓存，支持版本切换
🧠 智能补全
7 种插入模式 + 权重排序算法
📱 移动端优化
双击映射（空格→Tab, s→§, .→~），告别繁琐输入
📦 行为包导出
一键打包 ZIP，直接导入 Minecraft
🔁 行ID系统
撤销/重做时，光标位置可追溯
🧵 多线程加载
Grammar 在子线程编译，主线程流畅显示进度
