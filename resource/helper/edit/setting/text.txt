[title]高亮[/title]

编辑时，对应元素高亮的颜色，可以自己设置。

[title]补全[/title]

[b]最大补全数量：[/b]超过该阈值后面的补全项会被砍掉，以减少开销。

[b]显示提示词：[/b]开启后提示词会以括号的方式显示。如：
[code]cat   (猫)[/code]

[b]使用提示词加权：[/b]开启后，输入提示词能优先排在前面。由于要计算提示词的相似度，性能消耗翻倍。

[b]补全空间名称：[/b]关闭后，所有类似 [code]minecraft:apple[/code] 只显示 [code]apple[/code] 开启会带来少量的性能计算。
[title]行为[/title]

[b]双击映射：[/b]这是帮助符号的输入，如果你连续快速输入同一字符，就会进行转化。默认配置是如下：
[code]" " : "\t"
"s" : "§"
"." : "~"
"," : "^"
"z" : "="
"n" : "!"
"a" : "@"
"q" : "{"
"p" : "}"
"w" : "["
"o" : "]"
"e" : "<"
"i" : ">"[/code]

[title]自动保存[/title]

[b]自动保存：[/b]开启后允许自动保存。

[b]保存间隔：[/b]自动保存的时间间隔。

[title]语法[/title]

默认编辑器中的语法。

[b]语法路径：[/b]你可以查看语法的路径，每次启动时如果检测到没有目录，会尝试复制进去。
[b]提取语法：[/b]如果语法丢失或者是没有更新，可以手动更新。
